# 📊 Data Storyteller

An AI-powered data analysis tool that turns any CSV into instant insights.

## Features
- Upload any CSV file
- Auto-generated statistics and data preview
- Interactive filters for numeric and category columns
- 5 chart types — Scatter, Bar, Box, Histogram, Line
- AI-powered business insights using Groq LLM
- Download insights report

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Groq AI (LLaMA 3.3)

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key to `.env` file: `GROQ_API_KEY=your_key`
4. Run: `python -m streamlit run app.py`
